# coders-of-delhi
Coders of Delhi is a beginner-friendly Python project focused on logic building and core programming skills.
NumPy, Pandas & Web Scraping Practice Project (Python)

This repository contains hands-on Python practice covering NumPy, Pandas, and basic web scraping.
The project focuses on understanding data manipulation, analysis, and preprocessing using real and simulated datasets.

Key topics included:

NumPy arrays, indexing, slicing, broadcasting, and performance comparison

Pandas Series and DataFrames

Data filtering, sorting, querying, and aggregation

Data cleaning (missing values, duplicates, type conversion)

Reading external data and basic web scraping using requests

The code is written in a simple, step-by-step manner with comments for better understanding.
This project is ideal for beginners and data science aspirants who want practical exposure to Python-based data handling.

Tech Used: Python, NumPy, Pandas, Requests
